Display your Yahoo Status and your Yahoo Status Message on the widget

== Description ==
This plugin will get your yahoo status and yahoo status message and display it on the widget

Visit http://as247.vui360.com/blog/yahoo-status/ for more infomation

Specially thanks  S.C. Chen for A simple PHP HTML DOM parser



== Installation ==
1. Download the plugin and extract its content
2. Upload the complete folder `yahoo-status` to the `/wp-content/plugins/` directory
3. Chmod cookie.txt to 666
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Go to Settings->Yahoo Status to configure it
6. [Using](http://wordpress.org/extend/plugins/yahoo-status/other_notes/ "How to use yahoo status")

== Usage ==
1. Register new yahoo id.
2. Add your yahoo id to new yahoo id, please sure that your yahoo id in its friends list
3. Go to Setting->Yahoo Status: enter your yahoo id and new yahoo id infomation
	-Your yahoo id: your yahoo id
	-Temporary yahoo id: new yahoo id
	-Temporary yahoo password: new yahoo password

4. Go to Widget, drag and drop yahoo status widget to your sidebar
5. Visit [Yahoo Status home page](http://as247.vui360.com/blog/yahoo-status/ "Yahoo Status Home Page") for more

== Uninstall ==
Go to plugins management page and Deactive it.
